Why build a dedicated terminal?

Context Switching

Editor in one window, terminal in another, documentation in a browser. You lose flow state every time you Alt-Tab to check output.

Input Latency

Standard electron terminals often suffer from rendering lag. Forja uses xterm.js with WebGL acceleration for <16ms latency.

Session Amnesia

Closing your terminal usually kills your context. Forja persists workspaces, so Claude remembers exactly where you left off.

Native PTY Architecture

Built on top of node-pty and xterm.js, Forja behaves exactly like your native shell. Zsh, Fish, Bash—it all works out of the box with zero configuration required.

  • GPU Accelerated rendering
  • Full ANSI color support

Project-based Workspaces

Stop cd-ing into directories. Forja treats folders as first-class citizens. Switch between client, server, and infra repos instantly, preserving your shell history and environment variables.

Real-time Git Diff

When Claude suggests edits, see exactly what changed before you commit. The integrated diff view highlights additions and deletions directly alongside your terminal stream.
