# US-003 — Context Menu e Renomear Tabs de Terminal

**PRD:** PRD-003
**Sprint:** Fase 9

---

## US-003-1: Menu de Contexto nas Tabs

**Como** usuário do Forja,
**Quero** clicar com o botão direito em uma tab de terminal,
**Para** acessar ações de gerenciamento da tab.

### Critérios de Aceite

- [ ] Right-click em qualquer tab abre um context menu
- [ ] Menu exibe: "Edit tab" e "Fechar tab"
- [ ] "Fechar tab" tem estilo destrutivo
- [ ] Menu fecha ao pressionar Escape
- [ ] Menu fecha ao clicar fora

---

## US-003-2: Renomear Tab Inline

**Como** usuário do Forja,
**Quero** renomear uma tab de terminal inline,
**Para** identificar mais facilmente múltiplas sessões abertas.

### Critérios de Aceite

- [ ] Clicar "Edit tab" ativa modo de edição inline
- [ ] Input é pré-preenchido com o nome atual
- [ ] Enter confirma o novo nome
- [ ] Blur (clicar fora) confirma o novo nome
- [ ] Escape cancela sem alterar o nome
- [ ] Nome vazio ou whitespace não é aceito
- [ ] Nome customizado é exibido em vez do nome automático

---

## US-003-3: Fechar Tab pelo Menu de Contexto

**Como** usuário do Forja,
**Quero** fechar uma tab pelo menu de contexto,
**Para** ter uma forma alternativa ao botão X.

### Critérios de Aceite

- [ ] Clicar "Fechar tab" no menu fecha a tab
- [ ] A tab ativa é ajustada corretamente após fechar
