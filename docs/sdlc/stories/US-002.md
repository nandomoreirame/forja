# User Story US-002: Browser Screenshot to Clipboard

**Epic:** Browser Pane Enhancements
**PRD:** PRD-002
**Priority:** Medium
**Status:** In Progress

---

## Story

**As a** developer using Forja's embedded browser,
**I want to** capture a screenshot of the current webpage with a single click,
**So that** I can paste it into AI conversations or share it with my team without leaving the app.

## Acceptance Criteria

### AC-1: Camera Button Visibility
- **Given** the browser pane is open
- **When** I look at the browser toolbar
- **Then** I see a camera icon button next to (left of) the close button

### AC-2: Screenshot Capture
- **Given** the browser pane is showing a webpage
- **When** I click the camera button
- **Then** a screenshot of the webpage is captured

### AC-3: Clipboard Copy
- **Given** a screenshot was captured
- **When** the capture is complete
- **Then** the screenshot is available in my system clipboard (I can paste it anywhere)

### AC-4: Success Feedback
- **Given** the screenshot was successfully captured
- **When** the clipboard write is confirmed
- **Then** the camera button shows a green check state for ~2 seconds before reverting

### AC-5: Error Handling
- **Given** the screenshot capture fails (e.g., webview not loaded)
- **When** the error occurs
- **Then** the error is logged to console and the button reverts to normal state

## Tasks

- TASK-002-1: Add `browser:screenshot` IPC handler in `electron/main.ts`
- TASK-002-2: Expose `browser.screenshot()` in `electron/preload.cts`
- TASK-002-3: Add `screenshotBrowser()` function in `frontend/lib/ipc.ts`
- TASK-002-4: Add Camera button to `BrowserPane` toolbar
- TASK-002-5: Implement success feedback state (2s green)
- TASK-002-6: Write tests for all new functionality

## Notes

- Use `Camera` icon from `lucide-react` (consistent with existing icon usage)
- Button style matches other toolbar buttons: `h-7 w-7`, `rounded`, `text-ctp-overlay1`
- On success: briefly show `Check` icon or green color via state
- IPC pattern: `invoke("browser:screenshot", { webContentsId })`
