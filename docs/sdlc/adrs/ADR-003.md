# ADR-003 — Implementação de Context Menu e Inline Edit nas Tabs de Terminal

**Status:** Accepted
**Data:** 2026-03-10
**PRD:** PRD-003

---

## Contexto

Precisamos adicionar um menu de contexto nas tabs de terminal com opções de fechar e renomear. A renomeação deve ser inline (edição diretamente no texto da tab) sem abrir diálogos modais.

## Decisão

### 1. Context Menu: Radix UI `ContextMenu` (já disponível)

Usar o componente `ContextMenu` do shadcn/ui (via Radix UI), já presente em `frontend/components/ui/context-menu.tsx`.

**Alternativas consideradas:**
- **Dropdown Menu via botão:** Menos descobrível, não é o padrão UX para right-click
- **Menu nativo do Electron:** Fora do React render tree, difícil de estilizar com Catppuccin

### 2. Inline Edit: Reutilizar `InlineEdit` existente

O componente `InlineEdit` em `frontend/components/inline-edit.tsx` já implementa toda a lógica necessária (edição inline, Enter/Blur/Escape, validação). Reutilizá-lo é a escolha natural.

**Modificação necessária:** Adicionar prop `isEditing` externa para permitir que o context menu ative o modo de edição programaticamente.

### 3. Estado do nome customizado: `customName` no `TerminalTab`

Adicionar campo `customName?: string` na interface `TerminalTab` e action `renameTab(id, name)` no store.

**Integração com `computeTabDisplayNames`:** Tabs com `customName` retornam o nome customizado diretamente, sem aplicar numeração automática.

### 4. Estrutura do TabBar

Cada tab item será encapsulado em um `ContextMenu` wrapper. O estado `isEditing` será local ao item da tab (useState por tab), ativado pelo menu.

## Consequências

**Positivas:**
- Reutiliza componentes existentes (InlineEdit, ContextMenu)
- Estado simples, sem complicações
- UX familiar e acessível

**Negativas:**
- `InlineEdit` precisa de modificação para aceitar `isEditing` externo
- Leve aumento de complexidade no TabBar

## Implementação

- Modificar `TerminalTab` interface + `useTerminalTabsStore` (adicionar `customName`, `renameTab`)
- Modificar `computeTabDisplayNames` para respeitar `customName`
- Modificar `InlineEdit` para aceitar `isEditing` / `onEditingChange` props externas
- Modificar `TabBar` para usar `ContextMenu` por tab item
